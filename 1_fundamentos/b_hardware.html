Hardware

El hardware es un dispositivo físico que ejecuta algoritmos.

Unidad Central de Procesamiento
Central Processing Unit
CPU

  • Encargada de ejecutar un algoritmo.

  • Utiliza un par de señales eléctricas, conocidas como 1 y 0 para representar y procesar los algoritmos.

  • Al lenguaje a base de 1 y 0 de una computadora se le llama código de máquina y varía entr las distintas familias de CPU en el mercado.
