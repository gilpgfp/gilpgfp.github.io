Intérprete

  • Un intérprete ejecuta un código de un lenguaje de computadora sin generar externamente un archivo de compilación.

  • Los lenguajes de programación usados normalmente en un navegador web, como HTML, CSS y JavaScript son interpretados.

  • El lenguajes de programación Java se compila a un código de máquina genérica y luego es ejecutado por un intérprete de ese código.
