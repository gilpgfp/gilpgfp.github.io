Algoritmo

  • Un algoritmo es un conjunto ordenado de intrucciones que se ejecutan una tras otra para realizar una tarea.

  • El hardware es un dispositivo físico que ejecuta algoritmos.

  • El software son la ideas estructuradas que controlan el funcionamiento de una computadora,

  • Un lenguaje de programación es un conjunto de reglas precisas para indicar a una computadora como ejecutar un algoritmo.

  • Un compilador toma como entrada el código de un lenguaje de programación y genera el código para una computadora y sistema operativo precisos.

  • Un intérprete ejecuta un código de un lenguaje de computadora sin generar externamente un archivo de compilación.

  • Un pseudocódigo es la especificación de un algoritmo usando texto informal parecido al lenguaje humano.
