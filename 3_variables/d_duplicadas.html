Variables Duplicadas

Las variables o constantes declaradas en el mismo espacio no pueden tener el mismo nombre.

Ejemplo

<script>
debugger;
let a = "Q1";
document.write(a);
const a = "Q2";
document.write(a);
</script>

Consola

Uncaught SyntaxError: Identifier 'a' has already been declared
